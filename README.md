# RestFul API server
Example of restfull API server with jwt authetication

# Requirement
Please define the following in the .env file
`APPLICATION_PORT=`
`MONGODB_URL=`
`JWT_ISSUER=`
`JWT_CONSUMER=`
`JWT_SECRET=`

# Run the project
To run the project
1. Install the package by run the following command
`npm i`
2. After installing the packages. Run
`node index.js`
